Angular Version Highlights:

  • Angular v17

    • Deferrable views
    • Built-in control flow
    • Signals
    • New lifecycle hooks (afterRender, afterNextRender)
    • Vite and esbuild
    • Hydration enhancements

  • Angular v16

    • Standalone component migration and scaffolding
    • RxJS interoperability
    • Server-side rendering and hydration
    • Passing router data as component inputs
    • Flexible ngOnDestroy (DestroyRef)

  • Angular v15

    • Standalone APIs
    • Router and HttpClient tree-shakable standalone APIs
    • Directive composition API
    • Image directive (NgOptimized Image)
    • CDK Listbox
